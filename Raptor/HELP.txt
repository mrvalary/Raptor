Метод CheckCollision используется для проверки, пересекаются ли два прямоугольных объекта на экране. Он работает с двумя прямоугольниками, задаваемыми координатами их центров и размерами (шириной и высотой). Важно заметить, что этот метод проверяет, перекрываются ли два объекта, и возвращает true, если они пересекаются, и false, если нет.

Разбор метода
csharp
Копировать код
private bool CheckCollision(float x1, float y1, float width1, float height1, float x2, float y2, float width2, float height2)
{
    return x1 + width1 / 2 > x2 - width2 / 2 &&
           x1 - width1 / 2 < x2 + width2 / 2 &&
           y1 + height1 / 2 > y2 - height2 / 2 &&
           y1 - height1 / 2 < y2 + height2 / 2;
}
Параметры метода:
x1, y1 — координаты центра первого прямоугольника.
width1, height1 — ширина и высота первого прямоугольника.
x2, y2 — координаты центра второго прямоугольника.
width2, height2 — ширина и высота второго прямоугольника.
Логика работы:
Метод выполняет несколько проверок, чтобы удостовериться, что прямоугольники не находятся друг в друге:

x1 + width1 / 2 > x2 - width2 / 2

Проверка, что правая сторона первого прямоугольника не заходит за левую сторону второго прямоугольника.
x1 + width1 / 2 — правая граница первого прямоугольника (центр + половина ширины).
x2 - width2 / 2 — левая граница второго прямоугольника (центр - половина ширины).
x1 - width1 / 2 < x2 + width2 / 2

Проверка, что левая сторона первого прямоугольника не заходит за правую сторону второго прямоугольника.
x1 - width1 / 2 — левая граница первого прямоугольника (центр - половина ширины).
x2 + width2 / 2 — правая граница второго прямоугольника (центр + половина ширины).
y1 + height1 / 2 > y2 - height2 / 2

Проверка, что верхняя сторона первого прямоугольника не заходит за нижнюю сторону второго прямоугольника.
y1 + height1 / 2 — верхняя граница первого прямоугольника (центр + половина высоты).
y2 - height2 / 2 — нижняя граница второго прямоугольника (центр - половина высоты).
y1 - height1 / 2 < y2 + height2 / 2

Проверка, что нижняя сторона первого прямоугольника не заходит за верхнюю сторону второго прямоугольника.
y1 - height1 / 2 — нижняя граница первого прямоугольника (центр - половина высоты).
y2 + height2 / 2 — верхняя граница второго прямоугольника (центр + половина высоты).
Что происходит:
Каждое из условий проверяет, не пересекаются ли прямоугольники по одной из сторон.
Если все условия выполняются (то есть прямоугольники действительно перекрываются), метод возвращает true — значит, прямоугольники столкнулись.
Если хотя бы одно из условий не выполняется, метод возвращает false — значит, прямоугольники не перекрываются.
Пример:
Предположим, что у нас есть два прямоугольника:

Прямоугольник 1: центр в (0, 0), ширина 2, высота 2.
Прямоугольник 2: центр в (1, 1), ширина 2, высота 2.
Мы вызываем метод CheckCollision с этими параметрами:

csharp
Копировать код
CheckCollision(0, 0, 2, 2, 1, 1, 2, 2);
Проверки будут выглядеть так:

0 + 2 / 2 > 1 - 2 / 2 — 1 > 0, условие выполнено.
0 - 2 / 2 < 1 + 2 / 2 — -1 < 2, условие выполнено.
0 + 2 / 2 > 1 - 2 / 2 — 1 > 0, условие выполнено.
0 - 2 / 2 < 1 + 2 / 2 — -1 < 2, условие выполнено.
Так как все условия выполнены, метод вернёт true, что означает, что прямоугольники пересекаются.

Заключение:
Этот метод эффективно проверяет пересечение прямоугольников, используя простые математические операции для определения, перекрываются ли границы двух объектов.
